静态数组是在定义是就已经在栈上分配了空间大小，在运行时这个大小不能改变，如：
int a[10];
动态数组的大小是在运行是给定，即，运行时在堆上分配一定的存储空间，同时运行时还可以改变其大小，如：
int *a;
a = new int[10];
